Functions & Variable Environment

In this example we will have print like below

  • 10
  • 100
  • 1
As with every funtion a new execution context will be created so in each function we have variable with name x so that inner function variable value will be printed.
Example 1 
                var x= 1;
                a();
                b();
                console.log(x);
                function a() {
                    var x= 10;
                    console.log(x);
                }
                function b() {
                    var x= 100;
                    console.log(x);
                }
Example 2

with window we can acces global object so we can access all variables and function using window or this

                    var x= 1;
                    console.log("from window "+ window.x);
                    console.log("from this "+ this.x);
