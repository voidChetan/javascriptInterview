Cloures

Cloures is a combination of functions bundled together with a reference to its sorrounding state

A Cloures gives you access to an outer functions scope from inner function

Uses of clouser

  • Module Design Pattern
  • maintaining state
  • setTimoutes
  • Iterators
Example 1 
                    function a() {
                        var x = 10;
                        function b() {
                            console.log(x);
                        }
                        b();
                    }
                    a();
                    So here inner function b will have access to x as its in lexical scope of iots outer function
Example 2 
                    function top() {
                        var xxx= 999
                        function inner() {
                            console.log(xxx);
                        }
                        return inner; -> here we are returning function
                    }
                    var aaa = top(); - here we are storing whatever function top() is returning
                    aaa(); so if we execute aaa function it will print 999 as it will still hold the reference
Example 3 
                    function outer() {
                        var x =111;
                        function inner1() {
                            var y = 777;
                            function inner2() {
                                console.log(y,x);
                            }
                            inner2();
                        }
                        inner1();
                    }
                    outer();
                    with this also inner2 function will have 2 clouser one of inner1 and one is outer. so it will print 777,111
