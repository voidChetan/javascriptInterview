Javascript Prototype

It is just an object which is attached by javascript engine to array, object,variables or function whatever we create in javascript and we can acces many properties and functions available in object by using DOT operator

which just follows principle that everything in javascript is object for all object, array or function its base prototype is object prototype

Prototype inheritance 
                    let student = {
                        name:'Chetan',
                        city:'Nagpur',
                        getIntro : function() {
                            console.log(this.name + ' is From '+ this.city);
                        }
                    }
                    let student2 = {
                        name:"Ankita"
                    };
                    student2.__proto__ = student; =>here we are assigning prototype of student object to student2 object so it
                    can access getIntro function and city from student obj and name will be taken from student2 obj
                    console.log(student2.getIntro());
                    output=> Ankita is From Nagpur
